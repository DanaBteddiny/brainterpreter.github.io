Programming Languages

Python

  • Used for data processing, real-time communication, and interfacing with the Emotiv Insight headset.
  • Manages data flow from OSC (Open Sound Control) to the Raspberry Pi, handling command mapping and user interface display.
  • Ideal for its readability and vast library support for IoT applications.

C++

  • Powers the logic on the ESP32 microcontrollers, enabling efficient control of smart plugs and communication through ESP-NOW.
  • Optimized for low-level hardware interaction, ensuring fast execution and low memory usage.
  • Manages I2C communication between the Raspberry Pi and the ESP32 sender for real-time command transfer.
