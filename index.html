About

Empowering independence with Brainterpreter

Brainterpreter bridges the gap between thought and action, enabling individuals with severe disabilities to control their home environment using only their mind. Powered by brainwave detection, smart plugs, and seamless connectivity, it transforms mental commands into real-world actions

With just a thought, users can turn on lights, manage appliances, and interact with their surroundings independently. Brainterpreter redefines what is possible, making technology more accessible and life more connected.

How It Works

Brainwave Detection

The Emotiv Insight headset captures brainwave patterns and mental commands. These signals are processed and classified into specific actions, like "turn on," "turn off," or "select device."

Data Transmission to Raspberry Pi
  • The classified brainwave data is sent wirelessly from the headset to the Raspberry Pi using the EmotivBCI application and OSC protocol.
  • The Raspberry Pi receives the commands and maps them to corresponding smart home actions.
I2C Communication with ESP32 Sender

Commands are transmitted from the Raspberry Pi to an ESP32 sender module through a wired I2C connection for low-latency and secure communication.
Wireless Control Using ESP-NOW

The ESP32 sender module communicates wirelessly with multiple ESP32 smart plugs via ESP-NOW protocol, sending control signals without the need for Wi-Fi.
Real-Time Appliance Control

Smart plugs receive the commands and activate/deactivate connected appliances instantly, giving users seamless control over their environment.
Visual Feedback

A screen interface displays the status of each controlled device, providing real-time updates and visual confirmation of each action.
